# OceanAgendas


## Sobre
Criar um sistema web voltado para prestação de serviços por agendamento. A ideia inicial
é criar com as <b>linguagens Java + SpringBoot, Angular e banco de dados com Oracle</b>.


## Ideia Inicial
* Criar Projeto MVP ( Minimum Viable Product ).
* Implementar Dashboard com informações de dados para o usuario.
* Implementar Integração com WhatsApp e Telegram.
* Implementar IA com Sugestionador de cliente agendamento.
* Implementar exportação de dados csv.
